# Exemplos de Prompts

Este arquivo reúne exemplos de prompts utilizados ou aplicáveis ao laboratório sobre Microsoft Copilot, Azure OpenAI, criação assistida por IA, engenharia de prompts e filtros de conteúdo.

## Prompt para Resumir Texto

```text
Resuma o texto abaixo em até cinco tópicos. Mantenha apenas as ideias principais, use linguagem clara e destaque os conceitos mais importantes.

Texto:
[cole o texto aqui]
```

## Prompt para Explicar Conceito Técnico

```text
Explique o conceito de Azure OpenAI para uma pessoa iniciante em computação em nuvem. Use linguagem simples, inclua um exemplo prático e evite termos excessivamente técnicos.
```

## Prompt para Gerar Ideias

```text
Liste dez ideias de projetos simples que utilizem IA generativa para melhorar a produtividade de estudantes ou profissionais de tecnologia.

Para cada ideia, informe:
- Nome do projeto.
- Objetivo.
- Exemplo de uso.
```

## Prompt para Criar Planejamento

```text
Crie um plano de estudos de 7 dias sobre Inteligência Artificial Generativa.

O plano deve conter:
- Tema de cada dia.
- Objetivo de aprendizagem.
- Atividade prática.
- Sugestão de revisão.
```

## Prompt para Revisar Texto

```text
Revise o texto abaixo para deixá-lo mais claro, profissional e objetivo. Corrija problemas de gramática, melhore a organização das ideias e mantenha o sentido original.

Texto:
[cole o texto aqui]
```

## Prompt para Gerar Código Simples

```text
Gere um exemplo simples de código em JavaScript que receba uma lista de tarefas e retorne apenas as tarefas concluídas.

Inclua:
- Código comentado.
- Exemplo de entrada.
- Exemplo de saída.
```

## Observações sobre Como Melhorar Prompts

- Defina claramente o objetivo da solicitação.
- Informe o contexto necessário para orientar a resposta.
- Especifique o formato esperado, como lista, tabela, resumo ou passo a passo.
- Indique o público-alvo da resposta, como iniciante, técnico, gestor ou estudante.
- Inclua restrições quando necessário, como tamanho máximo, linguagem simples ou nível de detalhe.
- Revise a resposta gerada antes de utilizá-la em contextos profissionais.
- Evite inserir dados sensíveis, credenciais, chaves de API, tokens, senhas ou informações privadas.
- Quando necessário, refine o prompt com base na primeira resposta recebida.
